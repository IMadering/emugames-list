Sokoban for P/ECE

Sokoban is a franchise of puzzle video games in which the player pushes boxes around in a warehouse, trying to get them to storage locations.

Press SELECT on the title screen to select a level pack (the transferred PSD file) and press START to start the game.

To control the game, use the arrow keys to control the player, B to go back one turn, A to go back one turn, START to go back all turns, and SELECT to select levels.

In level selection mode, use the left and right arrow keys to select and START to play. Also, pressing SELECT in this mode will return you to the title screen.
